# 日馬連登録データ照合システム

日本馬術連盟（JEF）のシステムを利用して、お手元の出番表データ（Excel等）と登録情報の不一致を素早くチェック・修正するためのWebアプリです。
馬術競技会の準備等にお使いいただけます。  

## 特徴

- **Excelコピペ対応**: 出番表で記載される順番で 選手名、会員番号、馬匹名、登録番号 の4列をエクセルからそのままコピーし、またエクセルにそのまま貼り付けることができます。
- **一括照合**: サーバーサイドで日馬連のシステムへ問い合わせを行い、名称の相違や番号のミスを自動判定。
- **クイック修正**: 名前基準、または番号基準での修正候補を表示し、ボタン一つで入力データを上書き修正。

## 使い方

1. エクセル等から「選手名・会員番号・馬名・登録番号」の4列をコピーします。
2. 本アプリの入力エリアに貼り付けます。
3. 「一括照合を開始」ボタンをクリックします。
4. 入力データと日馬連登録情報が一致していない場合、状態列に「不一致」と表示されます。それらの行は、詳細列の「反映」ボタンを使ってデータを修正します。
5. 入力データが全て更新されたら改めて一括照合を行い、すべてが一致している状態にします。
6. 最後に「Excel用にコピー」ボタンを押し、エクセルに貼り付けして、出番表や結果表に利用します。

## 技術スタック

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
- **Backend**: Node.js (Vercel Serverless Functions)
- **Deployment**: Vercel
